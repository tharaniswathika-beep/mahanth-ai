# 🤖 Mahanth AI — Setup Guide

## ✅ STEP 1 — Paste Your Gemini API Key

1. Open the file called `.env` in Notepad
2. Replace `paste_your_gemini_api_key_here` with your actual key
3. It should look like:
   ```
   GEMINI_API_KEY=AIzaSyxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```
4. Save the file

---

## ✅ STEP 2 — Install Python (if not installed)

Download from: https://python.org/downloads
- ✅ Check "Add Python to PATH" during install

---

## ✅ STEP 3 — Run the App

### Windows (Easy way):
Double-click `START_WINDOWS.bat`

### Or open Command Prompt in this folder and run:
```
pip install -r requirements.txt
python app.py
```

---

## ✅ STEP 4 — Open in Browser

Go to: **http://localhost:5000**

Your Mahanth AI is ready! 🎉

---

## 🎤 Voice Input
- Works in **Google Chrome** only
- Click 🎤 button → speak → auto sends

## 🖼️ Image Upload
- Click 📎 button → choose image
- Ask any question about it

## 💬 Languages Supported
- English, Hindi, Telugu, Tamil, Kannada, and ALL world languages

---

## ❓ Troubleshooting

**"Error: API key not set"**
→ Open `.env` file and paste your Gemini API key

**"Cannot connect to server"**
→ Make sure `python app.py` is running

**Voice not working**
→ Use Google Chrome browser
